1 系统总览

本原型为「预约看房」功能的 PC 端后台管理系统，面向项目运营人员。覆盖从预约规则配置到数据管理的完整链路，包含 8 大功能模块。

▎核心业务流程

C 端用户入口
扫码进入小程序 → 填写表单 → 提交预约
PC 端管理入口
配置规则 → 管理记录 → 查看统计
2 预约须知设置

配置 C 端用户提交预约前必须阅读并勾选同意的须知内容。

▎功能元素说明

元素类型说明
富文本编辑器输入组件支持编辑预约须知内容（注意事项、携带证件、参观守则等）
实时预览区展示组件即时预览 C 端用户将看到的须知展示效果
保存按钮操作保存当前编辑的须知内容

▎交互逻辑

1 C 端用户提交预约前必须勾选同意《预约须知》复选框
! 未勾选直接提交 → Toast 阻断提示：「请阅读并勾选同意《预约须知》后方可提交」
→ 点击须知链接 → 展开须知弹窗（350ms 动画）展示完整内容
3 表单字段配置

配置 C 端预约表单中展示的字段及其属性。分为预置字段（不可删）和自定义字段（可增删改）。

▎预置字段清单

#字段名类型必填校验规则
1项目名称文本必填—
2客户姓名文本必填与名单中的姓名匹配
3客户电话文本必填11 位手机号格式
4人才编号文本必填与名单中的人才编号匹配
5看房日期日期必填在可选日期范围内
6看房时段下拉必填从已配置时段中选择，满员时段置灰

▎自定义字段属性

类型输入控件适用场景可设必填
文本单行输入框备注、地址、工号等自由文本✓
数字数字步进器年龄、工龄、数量等数值✓
日期日期选择器生日、入职日期等✓
下拉选项下拉单选性别、部门、意向户型等枚举值✓

▎交互规则

+ 点击「新增自定义字段」→ 弹窗选择类型、输入名称、设定必填
⚡ 每个字段独立开关控制 C 端是否显示
↕ 自定义字段拖拽排序（hover 显示拖拽手柄）
✕ 预置字段不可删除，仅可控制显隐
4 接待量与规则配置

配置可预约时段、接待量上限及各项预约限制规则。

▎规则清单

#规则项默认值说明拦截级别
1 时段接待量上限10 人/时段每时段最大可预约人数，满员后该时段不可再选阻断
2 最多提前预约天数7 天用户最多可提前 N 天预约，超出范围日期不可选阻断
3 ⚠ 至少提前预约天数1 天需至少提前 N 天预约（下限），当天不可预约阻断
4 取消预约截止时间到访日前一天 19:00超时不可取消，C 端取消按钮置灰或隐藏阻断
5 ✨ 陪同人数上限3 人每位预约客户最多可带 N 名陪同人员，超出需分批提示
6 重复预约限制30 天同一人才编号 30 天内仅允许 1 次有效预约阻断

▎C 端触发场景

时段满员弹窗提示已达上限，建议选择其他时段
超出提前天数弹窗提示最多提前 N 天
当天不可预约弹窗提示至少提前 N 天
30天重复弹窗提示已有记录，先取消原预约
超过取消截止详情页取消按钮不可用
5 资格验证设置

控制哪些人有资格进行预约，通过人员名单进行身份校验。

▎验证流程

▎名单管理操作

下载模板 — 获取标准 Excel 模板（含字段说明）
导入名单 — 批量导入 Excel，支持增量覆盖

6 签到签退设置

配置到访签到与离场签退的方式。预约码通过接口传给来访登记系统（运行在 iPad），用户到场后扫码核验。

▎签到方式

扫码签到

用户到现场使用微信扫描 iPad 来访登记系统中的预约二维码，系统自动校验预约状态与签到规则

▎签到判断流程

▎签退场景

用户离场时扫描 iPad 来访登记系统的签退二维码 → 打开C端签退页 → 点击签退后系统校验身份

▎时间规则与二维码

提前 时段开始前 15 分钟可签到，不足 → "非预约客户或未到预约时间"
迟到 距结束前 10 分钟停止签到，超时 → "预约时段即将结束，无法签到"
超时 未签到自动标记为「未到场」违约 → 锁定 N 天不可再次预约
约 预约码 · 来访登记系统生成 · 用户到场扫码核验
退 签退码 · 来访登记系统生成 · 离场扫码确认签退
7 消息通知设置

配置预约各环节的消息推送通知，每个节点可独立控制开关与渠道。

▎通知节点清单

#通知节点触发时机接收方内容
1 预约成功通知 用户提交预约后 用户 短信/微信通知，含预约详情（日期、时段、须知）
2 签到提醒通知 预约时段开始前 N 小时 用户 提醒按时到场签到（N 可在规则中配置）
3 新预约通知 有新预约提交时 管理员 通知管理员有新的预约记录待处理
4 取消预约通知 用户取消预约后 双方 同步通知管理员该预约已取消

▎配置能力

每个通知节点独立开关控制
支持配置通知渠道（短信/微信/站内）
通知模板内容可自定义
8 预约记录

查看和管理所有用户的预约记录，支持搜索、筛选、导出及状态流转管理。

▎列表字段

#字段类型说明
1预约编号文本系统自动生成的唯一标识
2客户姓名文本来自预约表单
3手机号文本脱敏展示中间 4 位
4人才编号文本身份校验关键字段
5看房日期日期用户选择的预约日期
6看房时段枚举用户选择的时段（如 09:00-10:00）
7陪同人数数字记录随行人员数量
8预约状态枚举见下方状态表
9预约时间时间戳用户提交预约的时间
10操作按钮组详情 / 取消（条件显示）

▎预约状态

状态标识色含义可执行操作
待到访待处理预约已提交，等待到访日查看详情 · 取消预约（未超期时）
已签到进行中用户已到场扫码签到查看详情
已签退已完成用户已离场扫码签退查看详情
已取消已结束用户主动取消或管理员取消查看详情
未到场违约超时未签到，自动违约标记查看详情

▎状态流转

▎功能操作

搜索 — 按姓名 / 手机号 / 人才编号模糊搜索
筛选 — 按日期范围、预约状态筛选
详情 — 弹窗展示完整预约信息（双列布局）
取消 — 二次确认弹窗（红色警告）仅在「待到访」且未超期时显示
导出 — 一键导出当前筛选结果为 Excel · 分页 — 默认每页 15 条，支持翻页跳转
